# React Router Project 🚀

This is a simple React project built with **Vite** and **React Router DOM v6**. It demonstrates client-side routing in a React application using modern best practices.

## 🛠️ Tech Stack

- ⚛️ React
- 🧭 React Router DOM (v6+)
- ⚡ Vite
- 💅 CSS (Custom styling)
